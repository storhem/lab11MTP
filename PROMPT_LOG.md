## Задание М1: Dockerfile для Python-приложения (FastAPI)

### Промпт 1
**Инструмент:** Claude Code

**Промпт:** "Сделай Dockerfile для Python-приложения — restnotes.py из ЛР6 (FastAPI). К работе необходимы файлы README, PROMPT_LOG и .gitignore. Репозиторий: https://github.com/storhem/LR6"

**Результат:** Claude изучил репозиторий ЛР6, получил содержимое restnotes.py. Создал:
- `python/restnotes.py` — скопирован из ЛР6
- `python/requirements.txt` — fastapi, uvicorn, httpx, pytest
- `python/Dockerfile` — двухстадийная сборка (builder → slim)
- `python/tests/test_notes.py` — 9 unit-тестов на все эндпоинты
- `.gitignore` — для Python, Go, Rust, Docker, IDE, ОС
- `README.md` — описание проекта и инструкции
- `PROMPT_LOG.md` — этот файл

**Что было решено автоматически:**
- Healthcheck без curl (используется `urllib.request` из стандартной библиотеки Python)
- Зависимости копируются через `/root/.local` для чистоты финального образа
- Тесты сбрасывают состояние через `autouse` фикстуру, чтобы не зависеть от порядка запуска
